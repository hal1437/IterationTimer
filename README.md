# ⏳ IterationTimer
<img width="600px" src="https://user-images.githubusercontent.com/8135472/142008962-ec745f8b-c6b6-4a82-8c5a-bdca69905239.png">

IterationTimerはゲームのスタミナなど、一定時間ごとに回復する値の予測値を表示するためのアプリです。

# 🧰 requirements
- 💎 ruby 2.7.3
- 🧵 bundler 2.1.4
- ⚒️ Xcode 13
- 🏭 xcodegen 2.18.0

# 🚪 installation
1. bundle install
1. xcodegen generate
